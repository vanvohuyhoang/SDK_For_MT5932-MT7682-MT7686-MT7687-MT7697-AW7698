Detailed Description

This structure defines the associated parameter type in the callback for BT_GAP_LE_PERIODIC_ADVERTISING_SYNC_ESTABLISHED_IND event.

Periodic advertising sync established indication, an event sent to the application once the periodic advertising sync is established. Application may send data periodically after receiving the periodic advertising sync established indication successfully.
