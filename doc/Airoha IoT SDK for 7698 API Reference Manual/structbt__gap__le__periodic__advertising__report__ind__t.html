Detailed Description

This structure defines the associated parameter type in the callback for BT_GAP_LE_PERIODIC_ADVERTISING_REPORT_IND event.

Periodic advertising report indication, an event sent to the application after receiving the periodic advertising report from the remote device. Create a periodic advertising sync to receive a periodic advertising report. Please refer to the bt_gap_le_periodic_advertising_create_sync() API.
