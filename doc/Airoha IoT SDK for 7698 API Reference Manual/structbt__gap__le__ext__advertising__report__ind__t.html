Detailed Description

This structure defines the associated parameter type in the callback for BT_GAP_LE_EXT_ADVERTISING_REPORT_IND event.

Extended advertising report indication, an event sent to the application after receiving the extended advertising report from the remote device. Enable the extended scan to receive an extended advertising report. Please refer to the bt_gap_le_set_extended_scan() API.
