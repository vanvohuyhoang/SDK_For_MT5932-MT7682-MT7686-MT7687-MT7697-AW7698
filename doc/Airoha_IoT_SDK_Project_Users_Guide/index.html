Introduction
Overview
  • This file guides you through the common and specific folder and file structure of available example projects. The example projects and applications are under [sdk_root]/project folder of the current SDK.
Common folders and files
  • The common folders and files in example projects are listed below.
    • readme.txt: The first document to guide you through using this project, including setup, usage and structure of this project.
    • EWARM: IAR IDE related files, including the compiler dependent startup assembly sources and the project file.
    • GCC: ARM GCC compiler related files, including the compiler dependent files and project makefiles.
    • inc: The header files of this project including the following common header files.
      • ept*: The configuration files generated by Easy Pinmux Tool (EPT), with the *.dws file under ept_config folder of a project or under the mtxxxx_hdk/template/ept_config folder for the common EPT file.(May not in hal_examples projects)
      • FreeRTOSConfig.h: FreeRTOS configuration. (Not in hal_examples projects)
      • hal_feature_config.h: The configuration file of HAL features.
      • memory_map.h: The memory layout symbol file.
      • sys_init.h: The header file of sys_init.c initializing the system. (Not in hal_examples projects)
    • MDK-ARM: Keil IDE related files, including the compiler dependent startup assembly sources and the project file.
    • src: The source files of this project.
Directory contents
  • Example projects using Airoha IoT SDK for Smart MCU: MT2523/MT2533
    • board/apps: Applications on Airoha IoT SDK for Smart MCU.
    • board/hal_examples: HAL reference examples without FreeRTOS on Airoha IoT SDK for Smart MCU.
    • board/templates: Templates for Smart MCU.
    • board/apps/iot_sdk_demo: A reference application with UI to demonstrate the basic functionalities of GNSS, Bluetooth, Sensor and watch.
  • Example projects using Airoha IoT SDK for WiFi: MT5932/MT7682/MT7686/MT7687/MT7697/AW7698
    • board/apps: Applications on Airoha IoT SDK for WiFi.
    • board/hal_examples: HAL reference examples without FreeRTOS on Airoha IoT SDK for WiFi.
    • board/templates: Templates for WiFi.
    • board/apps/iot_sdk_demo: A reference application to demonstrate the basic functionalities of Wi-Fi and internet protocols with command line interface (CLI).
  • Example projects using Airoha IoT SDK for BT Audio: AB155x
    • board/apps: Applications on Airoha IoT SDK for BT Audio
    • board/templates: Templates for BT Audio.
    • board/apps/headset_ref_design: A reference application to demonstrate the headset features.
