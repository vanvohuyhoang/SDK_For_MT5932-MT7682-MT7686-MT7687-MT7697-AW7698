Overview
  • This is not an example project. This project is a bootloader for AW7698 EVK. The developer can refer this project to understand how to build a bootloader.
Hardware and software environment
  • Environment configuration
    • The output logs are communicated through a micro USB cable to the PC from MK20 USB connector on the EVK.
    • Install the mbed serial driver according to the instructions at https://developer.mbed.org/handbook/Windows-serial-configuration. For more information, please refer to section "Installing the AW7698 EVK drivers on Microsoft Windows" on the "AW7698_EVK_Users_Guide.pdf" in [sdk_root]/doc/board/aw7698_evk folder.
    • Use a type-A to micro-B USB cable to connect type-A USB of the PC and MK20 micro-B USB connector on the AW7698 EVK. For more information about the connector cable, please refer to https://en.wikipedia.org/wiki/USB#Mini_and_micro_connectors.
    • Launch a terminal emulator program, such as Tera terminal on your PC for data logging through UART. For the installation details, please refer to section "Installing Tera terminal on Microsoft Windows" on the "Airoha IoT SDK for RTOS Get Started Guide" in [sdk_root]/doc folder.
    • COM port settings. baudrate: 115200, data bits: 8, stop bit: 1, parity: none and flow control: off.
Directory contents
  • Source and header files
    • main.c: Main program
    • inc/hal_feature_config.h: AW7698's feature configuration file.
    • src/custom_fota.c: Fota update function customization.
    • GCC/startup_bootloader.s: AW7698 startup file for GCC.
    • GCC/syscalls.c: The minimal implementation of the system calls.
    • EWARM/startup_aw7698.s: AW7698 startup file for IAR IDE.
  • Project configuration files using GCC
    • GCC/feature.mk: Feature configuration.
    • GCC/Makefile: Makefile.
    • GCC/bootloader.ld: Linker script.
  • Project configuration files using IAR IDE
    • EWARM/bootloader.ewp: IAR project file. Contains the project structure in XML format
    • EWARM/bootloader.eww: IAR workspace file. Contains project information.
    • EWARM/flash.icf: Linker script.
Run the example
  • Connect board to the PC with serial port cable.
  • Build bootloader and an example project with the command, "./build.sh aw7698_evk example_project_name bl" under the SDK root folder and download the binary file to AW7698 EVK.
  • Power on AW7698 EVK and bootloader log are showen on serial tool.
