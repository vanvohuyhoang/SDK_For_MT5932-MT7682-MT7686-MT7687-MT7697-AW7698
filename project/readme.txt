@par Overview
  - This file guides you through the common and specific folder and file
    structure of available example projects. The example projects and
    applications are under [sdk_root]/project folder of the current SDK.

@par Common folders and files
  - The common folders and files in example projects are listed below.
    - \b readme.txt:  The first document to guide you through using this
                      project, including setup, usage and structure of this
                      project.
    - \b EWARM:       IAR IDE related files, including the compiler dependent
                      startup assembly sources and the project file.
    - \b GCC:         ARM GCC compiler related files, including the compiler
                      dependent files and project makefiles.
    - \b inc:         The header files of this project including the following
                      common header files.
      - \b ept*:                  The configuration files generated by Easy
                                  Pinmux Tool (EPT), with the *.dws file under
                                  ept_config folder of a project or under the
                                  mtxxxx_hdk/template/ept_config folder for the
                                  common EPT file.(May not in hal_examples
                                  projects)
      - \b FreeRTOSConfig.h:      FreeRTOS configuration. (Not in hal_examples
                                  projects)
      - \b hal_feature_config.h:  The configuration file of HAL features.
      - \b memory_map.h:          The memory layout symbol file.
      - \b sys_init.h:            The header file of sys_init.c initializing
                                  the system. (Not in hal_examples projects)
    - \b MDK-ARM:     Keil IDE related files, including the compiler dependent
                      startup assembly sources and the project file.
    - \b src:         The source files of this project.

@par Directory contents
  - Example projects using Airoha IoT SDK for Smart MCU: MT2523/MT2533
    - \b board/apps:             Applications on Airoha IoT SDK for Smart MCU.
    - \b board/hal_examples:     HAL reference examples without FreeRTOS on
                                 Airoha IoT SDK for Smart MCU.
    - \b board/templates:        Templates for Smart MCU.
    - \b board/apps/iot_sdk_demo:
                                 A reference application with UI to
                                 demonstrate the basic functionalities of
                                 GNSS, Bluetooth, Sensor and watch.
  - Example projects using Airoha IoT SDK for WiFi: MT5932/MT7682/MT7686/MT7687/MT7697/AW7698
    - \b board/apps:             Applications on Airoha IoT SDK for WiFi.
    - \b board/hal_examples:     HAL reference examples without FreeRTOS on
                                 Airoha IoT SDK for WiFi.
    - \b board/templates:        Templates for WiFi.
    - \b board/apps/iot_sdk_demo:
                                 A reference application to demonstrate the
                                 basic functionalities of Wi-Fi and internet
                                 protocols with command line interface (CLI).
  - Example projects using Airoha IoT SDK for BT Audio: AB155x
    - \b board/apps:             Applications on Airoha IoT SDK for BT Audio
    - \b board/templates:        Templates for BT Audio.
    - \b board/apps/headset_ref_design:
                                 A reference application to demonstrate the
                                 headset features.
  * @}
